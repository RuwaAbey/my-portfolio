Group Activity Detection

Developed a dual-branch deep learning model leveraging CNNs and Transformers to detect group activities from skeleton coordinates and RGB video data. Implemented spatiotemporal feature fusion using MMAction2 and MMCV frameworks in PyTorch. Preprocessed pose data and video frames to effectively model human interactions, achieving robust activity recognition.

  • Model Development
  • Data Preprocessing

Private 5G Network Implementation

Designed and deployed a private 5G network using the Open Networking Foundation’s Aether platform. Configured the 5G core via Aether OnRamp and performed RAN emulation testing. Utilized Kubernetes clusters and Ansible playbooks for efficient core deployment and orchestration.

  • Kubernetes

Differential Amplifier

Designed a cascaded differential amplifier using SSM2213 matched transistor pairs with emitter degeneration for enhanced gain stability and frequency response. Achieved a DC gain of 13.2 and a 3dB bandwidth of 150 kHz through simulation and testing. Created the PCB in Altium Designer and analyzed differential and common-mode gain across a wide frequency range.

  • Circuit Simulation
  • PCB Designing

Hand Gesture Controlled 2DOF Robot Arm

Built a 2DOF robotic arm controlled by hand gestures using an MPU6050 sensor and ATmega328P microcontroller. Developed gesture recognition logic to generate precise PWM signals for servo motor control. Optimized power distribution with dual sources and a buck converter for efficient operation.

  • Robotics

Grip Force Sensor

Designed a force-sensing resistor (FSR) system for precise grip force measurement, featuring a 0.5–30 N range and 0.01 N accuracy. Utilized voltage divider and inverter circuits, applying a gradient method for force-resistance mapping and Kalman filtering for enhanced accuracy. Operates reliably at 22–30°C with a 0.9 cm sensor diameter.

  • Kalman FilteringCircuit Design
  • Circuit Design
  • Signal Processing

Security System for Pentagon

Developed a security system for a Pentagon Classified Document Room, integrating temperature, light, and floor pressure sensors for fire detection and anomaly monitoring. Implemented a dual-guard secret sequence entry mechanism and an emergency lockdown system. Controlled via Arduino with Python and PyFirmata, visualized using Node-RED.

  • Python
  • Node-RED
